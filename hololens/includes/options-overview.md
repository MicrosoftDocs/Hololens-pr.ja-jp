# [HoloLens 2 開発エディション](#tab/developeredition)

## 1. 開発エディションについて

> [!NOTE] 
> 開発エディションは、商用の分散ソリューションで展開することは許可されていません。 開発エディションは、商用設定のアプリケーションの開発およびテストを支援するために展開されることが許可されています。  

HoloLens 2 の開発エディションには、HoloLens 2、Azure クレジットの $500、Unity Pro の3ヶ月の試用、3ヶ月の PiXYZ プラグインの試用が含まれています。

| バンドル機能 | 説明 |
|---|---|
|  [Unity](https://unity.com/) | リアルタイムの3D 開発プラットフォーム。   |
|  [Unity 用 Pixyz プラグイン](https://www.pixyz-software.com/plugin/) | 開発環境でエンジニアリング CAD のワールドを接続するためのブリッジ &amp; です。   |
| [azure サービスの Azure クレジットの $500](https://azure.microsoft.com/resources/) | HoloLens 2 開発エディションに含まれている Azure クレジットは、Azure での mixed reality アプリケーションの構築とテストをすぐに開始することを目的としています。 |

すべての HoloLens 2 エディションには、同じハードウェアが搭載されています。この [ボックスに](../hololens2-hardware.md#hololens-components) は同一の [ソフトウェア](../hololens2-hardware.md#pre-installed-software)が付属しています。

## 2. クレジットを利用する

### Unity
固有のコードを受信します。 Unity Pro の試用版を使用するための手順: 
1. [Id.unity.com](http://id.unity.com/)にアクセス 
1. Unity ID でサインインするか、作成します。
1. [ **製品コードを使用する] をクリックする** 
1. 製品コードを入力し、[**製品コードの**使用] をクリックします。

[Unity ライセンスのライセンス認証を行う方法](https://support.unity3d.com/hc/articles/211438683-How-do-I-activate-my-license-)について詳しくは、こちらをご覧ください。 必要に応じて [Unity サポート](https://support.unity3d.com/hc) に連絡してください。  

### Pixyz
固有のコードを受信します。 Pixyz プラグインの試用版を利用するための手順: 
1. [Id.unity.com](http://id.unity.com/)にアクセス
1. [ **製品コードを使用する] をクリックする**
1. Pixyz プラグインの製品コードを入力し、[**製品コードの**使用] をクリックします。

[Pixyz ライセンスの償還](https://www.pixyz-software.com/documentations/html/2020.1/review/TrialLicense.html)方法について、詳細はこちらをご覧ください。 必要に応じて、 [Pixyz のサポート](https://www.pixyz-software.com/support/) に問い合わせてください。 

### Azure
ここでは、 [Mixed Reality の一般的な azure 製品](https://azure.microsoft.com/topic/mixed-reality/) をいくつか紹介します。そのためには、azure クレジットを利用することができます。
Azure クレジットの償還方法: $500 Azure クレジットを利用するには、Azure Pass のプロモーションコードを利用し、サブスクリプションをアクティブ化します。 [詳しい情報をご覧ください。](../hololens2-development-edition-faq.md#how-can-i-redeem-my-500-azure-credit)

- [Azure 空間アンカーを使用して、3D コンテンツのマッピング、共有、保持を行うアプリを構築する方法について説明します。](https://azure.microsoft.com/services/spatial-anchors/)
- [Azure リモートレンダリングが高品質でインタラクティブな3D コンテンツをレンダリングし、リアルタイムでデバイスにストリーミングする方法について説明します。](https://azure.microsoft.com/services/remote-rendering/)
