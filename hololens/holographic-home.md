<span data-ttu-id="d4f79-105">Windows PC のエクスペリエンスがデスクトップから始まるように、Windows Holographic では Mixed Reality ホームが最初に表示されます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-105">Just like the Windows PC experience starts with the desktop, Windows Holographic starts with mixed reality home.</span></span>  <span data-ttu-id="d4f79-106">Mixed Reality ホームでは、スタート メニューを使ってアプリ ウィンドウ、イマーシブ アプリ起動ツール、または 3D コンテンツを開いたり、配置したりすることができます。また、物理空間での配置を記憶しておくことができます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-106">Using the Start menu you can open and place app windows, immersive app launchers, and 3D content in mixed reality home, and their placement in your physical space will be remembered.</span></span>

## <span data-ttu-id="d4f79-107">スタート メニューを使用する</span><span class="sxs-lookup"><span data-stu-id="d4f79-107">Use the Start menu</span></span>

<span data-ttu-id="d4f79-108">HoloLens のスタート メニューは、アプリを開き、重要な状態情報を確認し、カメラなどのツールにアクセスする場所です。</span><span class="sxs-lookup"><span data-stu-id="d4f79-108">The Start menu on HoloLens is where you'll open apps, see important status info, and access tools like the camera.</span></span>

<span data-ttu-id="d4f79-109">HoloLens の使用中に**スタート ジェスチャ**を使うと、いつでも、どこからでもスタート メニューを開くことができます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-109">Wherever you are in HoloLens, you can always open the Start menu by using **Start gesture**.</span></span>  <span data-ttu-id="d4f79-110">HoloLens (第 1 世代) のスタート ジェスチャは[ブルーム](https://support.microsoft.com/help/12644/hololens-use-gestures)です。</span><span class="sxs-lookup"><span data-stu-id="d4f79-110">On HoloLens (1st gen) the Start gesture is [bloom](https://support.microsoft.com/help/12644/hololens-use-gestures).</span></span> <span data-ttu-id="d4f79-111">HoloLens 2 の[スタート ジェスチャ](hololens2-basic-usage.md#start-gesture)は、手首に表示される [スタート] アイコンのタップです。</span><span class="sxs-lookup"><span data-stu-id="d4f79-111">On HoloLens 2, the [Start gesture](hololens2-basic-usage.md#start-gesture) is to tap the Start icon that appears on your wrist.</span></span>  <span data-ttu-id="d4f79-112">音声を使って (「スタートに移動」と言う) スタート メニューを開くこともできます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-112">You can also open the Start menu using your voice by saying "Go to Start".</span></span>

> [!TIP]
> <span data-ttu-id="d4f79-113">開いているスタート メニューを閉じるには、スタート ジェスチャを使うか、スタート メニューを見て「閉じる」と言います。</span><span class="sxs-lookup"><span data-stu-id="d4f79-113">When the Start menu is open, use the Start gesture to close it, or look at the Start menu and say "Close".</span></span>

<span data-ttu-id="d4f79-114">スタート メニューの上部には、Wi-Fi、バッテリー、音量、時計の状態インジケーターが表示されます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-114">At the top of the Start menu, you'll see status indicators for Wi-Fi, battery, volume, and a clock.</span></span> <span data-ttu-id="d4f79-115">HoloLens 2 には、デバイスが音声認識に対応していて音声コマンドを聞き取っているかどうかを示すリスニング インジケーターもあります。</span><span class="sxs-lookup"><span data-stu-id="d4f79-115">On HoloLens 2 there is also a listening indicator that shows whether the device is speech enabled and is listening for voice commands.</span></span> <span data-ttu-id="d4f79-116">下部には、写真の撮影やビデオの録画に使用できる **[フォト]** ボタンおよび **[ビデオ]** ボタンがあります。</span><span class="sxs-lookup"><span data-stu-id="d4f79-116">At the bottom you'll find the **Photo** and **Video** buttons which allow you to take photos and video recordings.</span></span>  <span data-ttu-id="d4f79-117">また、見ているものを Miracast で他のデバイスに投影できる **[接続]** ボタンもあります。</span><span class="sxs-lookup"><span data-stu-id="d4f79-117">There is also a **Connect** button that allows you to project what you see to another device using Miracast.</span></span>

### <span data-ttu-id="d4f79-118">スタート メニューでアプリを見つける</span><span class="sxs-lookup"><span data-stu-id="d4f79-118">Find apps on Start menu</span></span>

<span data-ttu-id="d4f79-119">スタート メニューには、**[ピン留めされたアプリ]** と **[すべてのアプリ]** の一覧があります。</span><span class="sxs-lookup"><span data-stu-id="d4f79-119">The Start menu has a **Pinned apps** list and an **All apps** list.</span></span>

- <span data-ttu-id="d4f79-120">**[ピン留めされたアプリ]** の一覧には、ピン留めされているアプリが表示されます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-120">The **Pinned apps** list shows apps that have been pinned.</span></span> <span data-ttu-id="d4f79-121">アプリ タイルを**長押し**したときに表示されるコンテキスト メニューを使うと、**[ピン留めされたアプリ]** の一覧にアプリを追加することも、一覧からアプリを削除することもできます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-121">You can add and remove apps from the **Pinned apps** list using the context menu that appears when you **select and hold** on an app tile.</span></span>

- <span data-ttu-id="d4f79-122">**[すべてのアプリ]** の一覧には、デバイスにインストールされているすべてのアプリが表示されます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-122">The **All apps** list shows all apps that are installed on the device.</span></span>  <span data-ttu-id="d4f79-123">この一覧を表示するには、**スタート** メニューの右側にある **[すべてのアプリ]** ボタンを選択します。</span><span class="sxs-lookup"><span data-stu-id="d4f79-123">Select the **All apps** button on the right side of the **Start** menu to get to the list.</span></span>

<span data-ttu-id="d4f79-124">両方のアプリ一覧で、一覧内のすべてのアプリをページ表示で確認するには、スタート メニューの右側にある **PageUp** ボタンと **PageDown** ボタンを使用します。</span><span class="sxs-lookup"><span data-stu-id="d4f79-124">On both app lists, use the **Page up** and **Page down** buttons on the right side of the Start menu to page through all the apps in the list.</span></span>  <span data-ttu-id="d4f79-125">どちらのアプリ一覧でも、デバイス セッション中に最後に使用されたページが自動的に開きます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-125">Both app lists will automatically open to the page that was last used during a device session.</span></span>

> [!TIP]

### <span data-ttu-id="d4f79-128">スタート メニューからアプリを開く</span><span class="sxs-lookup"><span data-stu-id="d4f79-128">Open apps from Start menu</span></span>

<span data-ttu-id="d4f79-129">スタート メニューからアプリを開くには、**アプリ タイル**を**選択**します。</span><span class="sxs-lookup"><span data-stu-id="d4f79-129">To open an app from the Start menu, simply **select** an **app tile**.</span></span> <span data-ttu-id="d4f79-130">アプリの名前を言って開くこともできます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-130">You can also say the name of an app to open it.</span></span>

<span data-ttu-id="d4f79-131">スタート メニューからアプリを開くと、アプリがどのように設計されているかによって、次のいずれかが行われます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-131">When you open an app from the Start menu, one of the following will happen, depending on how the app is designed:</span></span>

- <span data-ttu-id="d4f79-132">**アプリ ウィンドウ**が配置されます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-132">An **app window** is placed.</span></span> <span data-ttu-id="d4f79-133">その後、アプリがウィンドウに読み込まれ、タッチ スクリーンと同じように使用できます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-133">The app is then loaded in the window and you can use it like a touch screen.</span></span>

<span data-ttu-id="d4f79-138">Mixed Reality ホームに配置されたアプリ ウィンドウとアプリ起動ツールは、削除しない限り維持されます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-138">App windows and app launchers placed in mixed reality home will stay around until you decide to remove them.</span></span>  <span data-ttu-id="d4f79-139">これにより、スタート メニューから再度開くことなく、アプリ ウィンドウを使用したり、イマーシブ アプリを起動したりできる便利なショートカットが提供されます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-139">They give you a convenient shortcut in the world to use those app windows or to launch immersive apps without having to open them again from the Start menu.</span></span> 

> [!NOTE]
><span data-ttu-id="d4f79-140">スマートフォンの場合と同様、HoloLens ではシステム リソースが自動的に管理されます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-140">Like on a phone, system resources is managed automatically on HoloLens.</span></span>  <span data-ttu-id="d4f79-141">たとえば、新しいイマーシブ アプリを開くと、実行中の他のアプリはすべて、直ちに非アクティブになります。</span><span class="sxs-lookup"><span data-stu-id="d4f79-141">For example, when you open a new immersive app, all other running apps will immediately become inactive.</span></span> <span data-ttu-id="d4f79-142">システム リソースを解放するために、Mixed Reality ホームでアプリ ウィンドウや起動ツールを削除する必要はありません。</span><span class="sxs-lookup"><span data-stu-id="d4f79-142">There is no need to remove app windows and launchers in mixed reality home to free up system resources.</span></span> 

## <span data-ttu-id="d4f79-143">HoloLens でアプリを使用する</span><span class="sxs-lookup"><span data-stu-id="d4f79-143">Using apps on HoloLens</span></span>

<span data-ttu-id="d4f79-144">HoloLens 上のアプリでは、アプリ ウィンドウ ビューまたはイマーシブ ビューを使用できます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-144">Apps on HoloLens can use app window view or immersive view.</span></span> <span data-ttu-id="d4f79-145">アプリ ウィンドウ ビューでは、アプリのコンテンツがウィンドウ内にを表示されます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-145">With app window view the app simply shows its content inside a window.</span></span> <span data-ttu-id="d4f79-146">イマーシブ ビューを使用するアプリでは、Mixed Reality ホームが非表示になり、ユーザーを取り巻く物理環境にコンテンツが表示されます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-146">With immersive view an app takes you away from mixed realty home where it can then display its content in the physical environment all around you.</span></span> <span data-ttu-id="d4f79-147">アプリで両方のビューを使用するように選択することもできます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-147">Apps can also choose to use both views.</span></span>

### <span data-ttu-id="d4f79-148">アプリ ウィンドウを使用する</span><span class="sxs-lookup"><span data-stu-id="d4f79-148">Use app windows</span></span>

<span data-ttu-id="d4f79-149">HoloLens (第 1 世代) のアプリ ウィンドウは、Mixed Reality ホームに配置され、使用されます。ここで、必要に応じてアプリ ウィンドウを[移動、サイズ変更、回転](hololens1-basic-usage.md#move-resize-and-rotate-apps)することもできます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-149">On HoloLens (1st gen) app windows are placed and used in mixed reality home, where you can [move, resize, and rotate](hololens1-basic-usage.md#move-resize-and-rotate-apps) them as you like.</span></span> <span data-ttu-id="d4f79-150">アプリ ウィンドウは、視線入力とジェスチャで使用する以外にも、Bluetooth で接続されたマウスとキーボードで操作することもできます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-150">In addition to using app windows with gaze and gesture, you can also use them with Bluetooth connected mouse and keyboard.</span></span>

<span data-ttu-id="d4f79-151">HoloLens 2 では、Mixed Reality ホームでアプリ ウィンドウを使用する以外にも、イマーシブ アプリ内でアプリ ウィンドウを (一度に 1 つずつ) 使用することもできます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-151">On HoloLens 2, in addition to using app windows in mixed reality home, you can also use one app window at a time inside an immersive app.</span></span> <span data-ttu-id="d4f79-152">アプリ ウィンドウを **[一緒に移動する]** モードにすることもできます。このモードでは、ユーザーが歩き回っても、アプリ ウィンドウが常にユーザーの目の前に表示されます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-152">You can also put an app window into **Follow me** mode where it will stay in front of you as you walk around.</span></span> <span data-ttu-id="d4f79-153">イマーシブ アプリ内でアプリ ウィンドウを開くと、自動的に **[一緒に移動する]** モードになります。</span><span class="sxs-lookup"><span data-stu-id="d4f79-153">When you open an app window while inside an immersive app, it will open in **Follow me** mode automatically.</span></span> <span data-ttu-id="d4f79-154">Mixed Reality ホームでもイマーシブ アプリ内でも、直接手を使ってアプリ ウィンドウを[移動、サイズ変更、および回転](hololens2-basic-usage.md#move-resize-and-rotate-holograms)できます。</span><span class="sxs-lookup"><span data-stu-id="d4f79-154">You can [move, resize, and rotate](hololens2-basic-usage.md#move-resize-and-rotate-holograms) app windows directly using your hands in both mixed reality home and inside an immersive app.</span></span>
