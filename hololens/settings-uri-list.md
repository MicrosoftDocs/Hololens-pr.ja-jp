# <span data-ttu-id="a744a-104">設定に関する URI</span><span class="sxs-lookup"><span data-stu-id="a744a-104">Settings URIs</span></span>

<span data-ttu-id="a744a-105">HoloLens デバイスの管理可能な機能の 1 つに、[Settings/PageVisibilityList ポリシー](https://docs.microsoft.com/windows/client-management/mdm/policy-csp-settings#settings-pagevisibilitylist)を使用した設定アプリ内で閲覧されるページの制限があります。</span><span class="sxs-lookup"><span data-stu-id="a744a-105">One of the manageable features for HoloLens devices is using the [Settings/PageVisibilityList policy](https://docs.microsoft.com/windows/client-management/mdm/policy-csp-settings#settings-pagevisibilitylist) to restrict the pages seen within the Settings app.</span></span> <span data-ttu-id="a744a-106">PageVisibilityList は、IT 管理者がシステム設定アプリの特定のページを表示またはアクセスできないようにしたり、または指定されたページ以外のすべてのページで同様に行うことを許可するポリシーです。</span><span class="sxs-lookup"><span data-stu-id="a744a-106">PageVisibilityList is a policy that allows IT Admins to either prevent specific pages in the System Settings app from being visible or accessible, or to do so for all pages except those specified.</span></span> 

> [!IMPORTANT]
> <span data-ttu-id="a744a-107">この機能は現在、[Windows Insider ビルド](hololens-insider.md)でのみ利用可能です。</span><span class="sxs-lookup"><span data-stu-id="a744a-107">This feature is currently only avalible in [Windows Insider builds](hololens-insider.md).</span></span> <span data-ttu-id="a744a-108">これに使用しようとしているデバイスをビルド 19041.1349+ にしてください。</span><span class="sxs-lookup"><span data-stu-id="a744a-108">Please ensure devices you intend to use this for are on build 19041.1349+.</span></span>
