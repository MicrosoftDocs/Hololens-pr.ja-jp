# <a name="hololens-1st-gen-release-notes"></a>HoloLens 第1世代 (gen) リリースノート

## <a name="hololens-1st-gen-long-term-servicing"></a>HoloLens (第1世代) 長期的なサービス
HoloLens (第1世代) が長期的なサービス (LTS) 状態に入りました。 今後の更新プログラムは、問題とセキュリティの修正に焦点を当てながら、HoloLens (第1世代) の Windows 10 Holographic、バージョン1809リリースの機能の同等性を維持します。

開発者にとっては、HoloLens (第1世代) アプリは OpenXR API をサポートしていません。  これらのヘッドセットは、unity 2019 LTS から2022までの完全なライフサイクルで、Unity 2019 LTS と WinRT API バックエンドで引き続きサポートされます。

### <a name="windows-10-version-1803-for-microsoft-hololens"></a>Microsoft HoloLens の Windows 10、バージョン1803

> **適用対象:** HoloLens (第1世代)

Windows 10 バージョン1803は Windows 10 バージョン1607のリリース以降に Windows Holographic for Business する最初の機能更新プログラムです。 この更新プログラムでは、次の変更が導入されています。

- 以前は、デバイスで [VPN が利用可能] オプションであるかどうかを確認することで、商用スイートのアップグレードライセンスが HoloLens デバイスに適用されていることのみを確認できました。 これで、  >  アップグレードライセンスが適用された後 **Windows Holographic for Business** 設定 **システム** に表示されるようになります。 [Windows Holographic for Business 機能のロックを解除する方法について説明](hololens1-upgrade-enterprise.md)します。

- オペレーティングシステムのビルド番号は、エクスプローラーアプリおよび[Windows デバイス回復ツール (wdrt)](https://support.microsoft.com/help/12379/windows-10-mobile-device-recovery-tool-faq)の [デバイスのプロパティ] で確認できます。
- Windows 構成デザイナーツールの新しい **デバイスの HoloLens プロビジョニング** ウィザードを使用すると、HoloLens デバイスのプロビジョニングがより簡単になりました。 ウィザードでは、セットアップエクスペリエンスとネットワーク接続の構成、開発者モードの設定、一括 Azure AD トークンの取得を行うことができます。 [HoloLens に簡単なプロビジョニングウィザードを使用する方法について説明](hololens-provisioning.md#provisioning-package-hololens-wizard)します。

- プロビジョニングパッケージでローカルアカウントを作成すると、42日おきにパスワードの有効期限が切れなくなります。

- [HoloLens は、シングルアプリまたはマルチアプリキオスクとして構成](hololens-kiosk.md)できます。 マルチアプリキオスクモードでは、指定したアプリのみを実行するように HoloLens を設定し、ユーザーが変更できないようにします。

- メディア転送プロトコル (MTP) を有効にすると、USB で HoloLens デバイスを pc に接続し、HoloLens と pc 間でファイルを転送できるようになります。 また、ファイルエクスプローラーアプリを使用して、HoloLens 内からファイルを移動したり削除したりすることもできます。

- 以前は、Azure Active Directory (Azure AD) アカウントを使用してデバイスにサインインした後、会社のリソースにアクセスするには **設定** に **職場のアクセス権を追加** する必要がありました。 これで、Azure AD アカウントを使用してサインインすると、自動的に登録が行われます。

- サインインする前に、[パスワード] フィールドの下にあるネットワークアイコンを選択して、接続先となる別の Wi-Fi ネットワークを選択できます。 また、ホテル、カンファレンスセンター、企業などのゲストネットワークに接続することもできます。

- Azure AD アカウントを使用して、[複数のユーザーと HoloLens](hololens-multiple-users.md)を簡単に共有できるようになりました。

- セットアップまたはサインインに失敗した場合は、[新しい **収集情報** ] オプションを選択して、トラブルシューティングの診断ログを取得します。

- 個々のユーザーは、モバイルデバイス管理 (MDM) にデバイスを登録しなくても、会社の電子メールを同期できます。 Microsoft アカウントでデバイスを使用し、メールアプリをダウンロードしてインストールし、電子メールアカウントを直接追加することができます。

- **設定**  >  **アカウント** が  >  **職場または学校** の情報にアクセスするときに、デバイスの MDM 同期の状態を確認でき  >  ます。 [ **デバイスの同期の状態** ] セクションでは、同期の開始、MDM によって管理されている領域の確認、高度な診断レポートの作成とエクスポートを行うことができます。
