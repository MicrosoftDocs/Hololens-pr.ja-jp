# <a name="get-your-hololens-2-ready-to-use"></a>HoloLens 2 の使用準備

以下の手順では、初めて HoloLens 2 をセットアップする方法について説明します。

## <a name="charge-your-hololens"></a>HoloLens を充電する

USB-C ケーブル (付属) を使って、電源アダプターを充電ポートに接続します。 電源アダプターは電源コンセントに差し込みます。 デバイスに付属している AC アダプターと USB ケーブル (C to C) は、HoloLens 2 を充電する最適な方法です。 充電器は、18W の電力 (2A で 9V) を供給します。 付属のプラグを使用すると、HoloLens 2 デバイスは、デバイスがスタンバイ状態のときに 65 分未満でバッテリーを完全に充電できます。

充電率と速度は、デバイスが実行されている環境によって異なる場合があります。

- デバイスの充電中は、バッテリー インジケーターが点灯し、現在の充電レベルを示します。  最後のライトはゆっくりと点滅し、充電中であることを示します。
- HoloLens の電源が入っているときは、バッテリー インジケーターにバッテリー レベルが段階的に表示されます。
- 5 個のライトのうち 1 個しか点灯していない場合は、バッテリー レベルが 20 パーセントを下回っています。
- バッテリー レベルが極端に低い場合にデバイスの電源を入れようとすると、1 個のライトが短く点滅してから消えます。

デバイス充電[ についてより詳細な情報が必要な場合は、](hololens2-charging.md#charging-the-device)こちらを参照してください。 

## <a name="adjust-fit"></a>フィット調整

HoloLens 2 を頭に装着します。 眼鏡をかけている場合は、かけたままで問題ありません。  額にブロー パッドが適度に密着し、バック バンドが後頭部の中心に来るようにします。

必要に応じて調整ノブを回してヘッドバンドを広げ、オーバーヘッド ストラップを緩めます。

![HoloLens 2 のフィット調整](images/hololens2-fit.png)

### <a name="attach-and-detach-the-overhead-strap"></a>オーバーヘッド ストラップの取り付けと取り外し

オーバーヘッド ストラップは必須ではありませんが、これを使用すると、HoloLens 2 を長時間装着する場合の快適性が向上します。

オーバーヘッド ストラップの前面部分を取り外すには、ストラップのフックを外し、ブロウ パッドの格納式ループから引き抜きます。 もう一度取り付けるには、ループを引き出し、ストラップを元のように通します。

オーバーヘッド ストラップの背面部分を取り外すには、各接続タブの下にあるボタンを押してそっと引っ張ります。 もう一度取り付けるには、接続タブを元のスロットにカチッと差し込みます。

![HoloLens 2 ヘッド ストラップの取り付けまたは取り外し](images/hololens2-headstrap.png)

## <a name="turn-on-the-hololens-2"></a>HoloLens 2 の電源を入れる

HoloLens 2 の電源を入れるには、電源ボタンを押します。  電源ボタンの下にある LED は、バッテリー レベルを示します。

> [!NOTE]
> HoloLens 2 を箱から取り出し、初めて電源を入れる場合は、電源ボタンを 4 秒以上押し続けてオンにします。 次に HoloLens 2 の電源を入れる際には、電源ボタンを短く押すだけで起動できます。

### <a name="power-button-actions-for-different-power-transitions"></a>さまざまな電源状態での電源ボタンの操作

| 目的 | このアクションを実行 | HoloLens 2 の動作 |
| - | - | - |
| 電源を入れる | ボタンを 1 回押します。 | 5 つのライトがすべて点灯してから、バッテリー レベルを反映して変化します。 4 秒後に、サウンドが再生されます。 |
| スリープ状態にする | ボタンを 1 回押します。 | 5 つのライトがすべて点灯し、1 つずつゆっくり消えます。 すべてのライトが消えると、サウンドが再生され、画面に終了メッセージが表示されます。 |
| スリープ状態から復帰させる | ボタンを 1 回押します。 | 5 つのライトがすべて点灯してから、バッテリー レベルを反映して変化します。 すぐにサウンドが再生されます。 |
| 電源を切る | 5 秒間押し続けます。 |  5 つのライトがすべて点灯し、1 つずつゆっくり消えます。 すべてのライトが消えると、サウンドが再生され、画面に終了メッセージが表示されます。 |
| Hololens が応答しない場合に強制的に再起動する | 10 秒間押し続けます。 | 5 つのライトがすべて点灯し、1 つずつゆっくり消えます。 すべてのライトが消えます。 |

## <a name="hololens-behavior-reference"></a>HoloLens の動作リファレンス

HoloLens のインジケーター ライトの意味がわからない場合 充電中に HoloLens がどのように動作するかご興味をお持ちですか?  ヘルプ情報は次のとおりです。

### <a name="charging-behavior"></a>充電動作

| デバイスの状態 | アクション | HoloLens 2 で実行される処理 |
| - | - | - |
| OFF | USB ケーブルを接続する | デバイスがオンになり、バッテリー レベルを示すインジケーター ライトが点灯し、デバイスが充電を開始します。
| ON | USB ケーブルを取り外す | デバイスの充電が停止する
| ON | USB ケーブルを接続する | デバイスが充電を開始する
| スリープ | USB ケーブルを接続する | デバイスが充電を開始する
| スリープ | USB ケーブルを取り外す | デバイスの充電が停止する
| USB ケーブルを接続した状態でオン | デバイスをオフにする | デバイスがオンになり、バッテリー レベルを示すインジケーター ライトが点灯し、デバイスが充電を開始する |

### <a name="lights-that-indicate-the-battery-level"></a>バッテリー レベルを示すライト

| ライトの数 | バッテリー レベル |
| - | - |
| 4 つのライトが点灯、1 つのライトがゆっくり点滅 | 100% から 81% (完全充電) |
| 3 つのライトが点灯、1 つのライトがゆっくり点滅 | 80% と 61% の間 |
| 2 つのライトが点灯、1 つのライトがゆっくり点滅 | 60% と 41% の間 |
| 1 つのライトが点灯、1 つのライトがゆっくり点滅 | 40% と 21% の間 |
| 1 つのライトがゆっくり点滅 | 20% から 5% 以下 (バッテリー切れ) |

### <a name="sleep-behavior"></a>スリープ動作

| デバイスの状態 | アクション | HoloLens 2 で実行される処理 |
| - | - | - |
| ON | 電源ボタンを 1 回押す | デバイスがスリープに移行し、すべてのインジケーター ライトがオフになる |
| ON | 3 分間操作なし | デバイスがスリープに移行し、すべてのインジケーター ライトがオフになる |
| スリープ | 電源ボタンを 1 回押す | デバイスがオンになり、インジケーター ライトが点灯する |

### <a name="lights-to-indicate-problems"></a>問題を示すライト

| ユーザーによる操作 | ライトの動作 | 説明 |
| - | - | - |
| 電源ボタンを押します。 | 1 つのライトが 5 回点滅してから消えます。 | HoloLens のバッテリー残量が極端に少なくなっています。 HoloLens を充電します。 |
| 電源ボタンを押します。 | 5 つのライトがすべて 5 回点滅してから消えます。 |  HoloLens を正しく起動できません。エラー状態になっています。 [オペレーティング システムを再インストールして](hololens-recovery.md)、デバイスを回復します。 |
| 電源ボタンを押します。 | 1 番目、3 番目、および 5 番目のライトが連続して同時に点滅します。 |  HoloLens にハードウェア障害が発生している可能性があります。 [サポート](https://support.microsoft.com/en-us/supportforbusiness/productselection?sapid=3ec35c62-022f-466b-3a1e-dbbb7b9a55fb)に問い合わせます。 |

## <a name="safety-and-comfort"></a>安全性と快適性

### <a name="use-hololens-in-safe-surroundings"></a>HoloLens を安全な環境で使用する

HoloLens は、障害物やつまずく危険性のない安全な場所でお使いください。 乗り物の運転中や、その他危険を伴う可能性のある行為を行っているときなど、視野に障害物があってはならない場合や、細心の注意を払うことができない場合は、使用しないでください。

### <a name="stay-comfortable"></a>快適性を保つ

HoloLens を使用する最初の数セッションは短時間にとどめ、必ず休憩を取りましょう。 不快感を覚えた場合は使用を中止し、気分が良くなるまで休んでください。 考えられる症状には、一時的な吐き気、乗り物酔い、目まい、方向感覚障害、頭痛、疲労、疲れ目、ドライ アイなどがあります。

[「製品の安全性に関する警告と手順」](https://support.microsoft.com/help/4558037/product-safety-warnings-and-instructions)を参照してください。
