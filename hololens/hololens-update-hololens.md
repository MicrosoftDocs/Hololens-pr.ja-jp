# <a name="update-hololens-2"></a>更新HoloLens 2

## <a name="overview"></a>概要

新機能、バグ修正、およびセキュリティ更新プログラムに常に取り組み中です。 これらの更新プログラムの準備ができたら、通知されます。

設定に基づいて、HoloLens は、電源に接続され、インターネットに接続され、スタンバイ状態になるたびに、システム更新プログラムを自動的にダウンロードしてインストールします。

お使HoloLens常に更新されていることを確認するには、付属のプラグを差し込んでおいたままにしてください。 また、インターネットHoloLens接続する必要があります。 これにより、システム更新プログラムが自動的にダウンロードおよびインストールされます。 

Windows Update サービスを使用すると、更新プロセスの複数の側面 (どのデバイスが何時にどの更新プログラムを取得するのかなど) を制御できます。 このコントロールは、テスト用にデバイスの一部に更新HoloLens展開できるので便利です。 次に、残りの更新プログラムをロールアウトします。 または、更新プログラムの種類によって、異なる更新スケジュールを定義することもできます。

## <a name="types-of-updates"></a>更新プログラムの種類

このHoloLens、2 種類の更新プログラムを自動的に管理できます。

- 機能更新プログラム: 年に 2 回リリースされます。
- 品質更新プログラム: 重要なセキュリティ更新プログラムを含める。 毎月、または必要に応じてリリースされます。

**Update** / **AllowAutoUpdate を使用** して、更新プログラムのスキャン、ダウンロード、インストールを管理します。 

## <a name="scheduling-updates"></a>更新プログラムのスケジュール設定

更新スケジュールを設定できます。 特定の日、または毎日、特定の時刻に指定できます。 たとえば、午後 5 時、またはアクティブ時間外です。

最後に、更新戦略の計画に関するいくつかの言葉を紹介します。 Microsoft では更新プログラムの遅延がサポートされています。そのため、Microsoft が更新プログラムをリリースしてデバイスにインストールするまでの待機時間を決定できます。

会社では、すべての新機能を最初に試して、すべてが機能し、サポート チームが準備するために新しい更新プログラムに精通している場合があります。 すべてが良いことを確認したら、会社全体に更新プログラムをロールアウトします。 デバイスのサブセットを更新リングと呼ばれるさまざまな遅延ポリシーに関連付け、組織の更新プログラムのロールアウト戦略を調整できます。

### <a name="check-for-updates-and-manually-update"></a>更新プログラムを確認し、手動で更新する

設定では、いつでも更新プログラムを確認できます。  利用可能な更新プログラムを表示し、新しい更新プログラムを確認するには:

1. **[設定]** アプリを開きます。
1. [Update **& Security Windows**  >  **に移動します**。
1. **[更新プログラムの確認]** をクリックします。

更新プログラムが利用可能な場合は、新しいバージョンのダウンロードが開始されます。 ダウンロードが完了したら、[今すぐ再起動 **] ボタンを** 選択してインストールをトリガーします。 デバイスが 40% を下回り、接続されていない場合、再起動によって更新プログラムのインストールは開始されません。

更新プログラムHoloLensインストール中に、回転する歯車と進行状況インジケーターが表示されます。 この期間中は、HoloLensをオフにしてください。 インストールが完了すると、自動的に再起動されます。

HoloLens一度に 1 つの更新プログラムが適用されます。  最新バージョンHoloLensバージョンが複数ある場合は、更新プロセスを複数回実行して完全に最新の情報に更新する必要があります。
