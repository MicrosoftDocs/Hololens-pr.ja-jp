# <a name="hololens-2-development-edition-faq"></a>HoloLens 2 Development Edition に関してよく寄せられる質問

## <a name="general-questions"></a>一般的な質問

### <a name="what-do-i-get-with-the-hololens-2-development-edition"></a>HoloLens 2 Development Edition では、どのようなことができますか。

HoloLens 2 の開発エディションには、HoloLens 2、$500 for Azure クレジット、3か月の Unity Pro 試用版、3か月の Pixyz プラグイン試用版が含まれています。

### <a name="what-is-the-difference-between-hololens-2-and-hololens-2-development-edition"></a>HoloLens 2 と HoloLens 2 の開発エディションの違いは何ですか。

HoloLens 2 Development Edition のみを使用して、運用環境設定で HoloLens 2 (商用エディション) と共に使用する内部エンタープライズアプリケーションを構築およびテストすることができます。 Microsoft では、開発エディションの購入の一環として、その裁量で、チェックアウトプロセス中に特定の権利を提供することがあります。 これらの権利には、特定の割引クレジットまたは特定のマイクロソフトまたはサードパーティのサービスやツールへの試用アクセスを限定された期間に利用する権利が含まれます。 [詳細情報](hololens2-options-dev-edition.md)

### <a name="can-any-edition-of-hololens-2-be-used-to-develop"></a>HoloLens 2 の任意のエディションを開発に使用できますか。

はい。 HoloLens 2 の任意のエディションを使用して、アプリを開発できます。

### <a name="how-and-will-i-receive-my-500-in-azure-credits-3-month-unity-pro-trial-and-3-month-pixyz-plugin-trial"></a>Azure クレジット、3か月の Unity Pro 試用版、3か月の Pixyz プラグイン試用版で $500 を受け取るにはどうすればよいですか?

割引クレジットまたは試用版へのアクセスは、2営業日以内に購入された電子メールに送信されます。

### <a name="can-i-upgrade-my-hololens-2-device-only-device-to-the-hololens-2-development-edition-to-get-the-trial-credits"></a>"HoloLens 2 (デバイスのみ)" デバイスを HoloLens 2 開発エディションに "アップグレード" して、試用クレジットを取得することはできますか。

いいえ、デバイス転送プランは提供されていません。 任意のエディションから他のエディションに転送することはできません。

### <a name="what-is-unity-pro"></a>Unity Pro とは

Unity Pro は、小規模/中規模のスタジオ/企業向けのリアルタイム3D プラットフォームです。 技術的なスキルと技術面以外のスキルを組み合わせたチーム向けに設計されています。 [詳細情報](https://store.unity.com/products/unity-pro)

### <a name="what-is-pixyz-plugin"></a>Pixyz プラグインとは

Pixyz プラグインは、ほぼすべての形式を読み取って変換し、Unity に最適化して置き、リアルタイム アプリケーションにすぐに使用できる資産を提供するためにそれをステージします。 [詳細情報](https://unity.com/products/pixyz)

### <a name="is-financing-for-the-hololens-2-development-edition--available"></a>HoloLens 2 Development Edition は利用できますか?

はい [。HoloLens 2 Development Edition](https://ms.liftforward.com/hololens?product=hololens-development-edition) の場合は、米国 でしか使用米国。 

### <a name="who-do-i-contact-with-questions-about-my-hololens-2-development-edition-entitlements"></a>開発エディションのエンタイトルメントに関する質問HoloLens 2問い合わせは誰ですか?

権利に関する質問がある場合は、営業スペシャリスト (866-425-4709、月から金、午前 6 時から午後 6 時、PST) にお問い合わせください。

### <a name="can-my-company-buy-multiple-hololens-2-development-edition-devices"></a>会社は、Development Edition デバイスHoloLens 2購入できますか? 

このデバイスは、Mixed Reality ソリューションを構築する開発者向けです。 顧客トランザクションごとに 1 つの購入を制限します。 会社が複数の Development Edition デバイスを購入する場合は、会社に代わって HoloLens 2 Development Edition デバイスを購入するために個別のトランザクションが必要になります。 

### <a name="can-i-show-my-hololens-2-development-edition-device-to-my-customers"></a>お客様に HoloLens 2 Development Edition デバイスを表示できますか? 

はい。このデバイスは、Mixed Reality ソリューションのプロトタイプ作成とテストに使用することを目的としています。

MRTK-Unreal は、最初にリリースされる UX ツールコンポーネントであり、現在は HoloLens 2 でのみサポートされています。 コンポーネント プラグインには、入力シミュレーション、ハンド インタラクション アクター、押しボタン コンポーネント、マニピュレーター コンポーネント、動作追従コンポーネントに対応した一般的な UX 機能のコード、ブループリント、サンプル資産が含まれています。 [詳細情報](https://microsoft.github.io/MixedReality-UXTools-Unreal/version/public/0.9.x/Docs/WelcomeToUXTools.html)
